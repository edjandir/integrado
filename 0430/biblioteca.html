🛠️ Desafio Prático: Gestor de Equipe
Crie uma lista chamada equipe. Cada item da lista deve ser um dicionário contendo: "nome" e "habilidade".
  1. Adicione pelo menos 3 integrantes à lista (ex: Tech Lead, Designer, Dev).
  2. Peça ao usuário uma nova habilidade para o primeiro integrante da lista e atualize o dicionário dele.
  3. Use um loop for para imprimir o nome e a habilidade de todos os integrantes formatado como uma lista de crachás.
